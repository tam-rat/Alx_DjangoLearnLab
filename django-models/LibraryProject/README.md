\# LibraryProject



This is a basic Django project created as part of the Introduction to Django task.



\## Objective

To gain familiarity with Django by setting up a Django development environment, creating a project, and running the development server.



\## How to Run the Project



1\. Install Django:

&nbsp;  pip install django



2\. Navigate to the project directory:

&nbsp;  cd LibraryProject



3\. Start the development server:

&nbsp;  python manage.py runserver



4\. Open your browser and visit:

&nbsp;  http://127.0.0.1:8000/



You should see the default Django welcome page.
